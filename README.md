# Portfolio Website

This repo is a ready-to-deploy single-page portfolio built with **Next.js + TypeScript + Tailwind**.

## Deploy in 2 minutes (No coding needed)

### Option A — Deploy on Vercel (Recommended)
1. Click **Deploy** button below
2. Sign in with GitHub
3. Click **Deploy**
4. Done ✅

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

---

## Update Your Content
Your data is stored here:

**`/data/draft.json`**

Edit that file and redeploy.

---

## Run locally (optional)
```bash
npm install
npm run dev
````

Open: [http://localhost:3000](http://localhost:3000)
